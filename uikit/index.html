LABEL

af-label

Labels implement a read-only text view. You can use this class to draw one or multiple lines of static text, such as those you might use to identify other parts of your user interface. Label provides support for both simple and complex styling of the label text.

Parameters

Type label-type

  • header Header
  • title Title
  • subheader Subheader
  • body-1[default] Body Type 1
  • body-2 Body2 Type 2
  • caption Caption
  • label Label

Color color

  • ${hexColor} Hex Colors
  • red|orange|yellow|olive|green|teal|blue|violet|purple|pink|brown|grey|black Color Names
  • primary|secondary Theme Colors

Dim dim

Text text

  • ${textVar} calculated
  • val constant

Samples

Red Header

af-label(color="red" label-type="header" text="Header")

Dark Subheader

af-label(color="#000000" label-type="subheader" text="Sub Header")

Dim Caption

af-label(label-type="caption" dim text="Caption")
